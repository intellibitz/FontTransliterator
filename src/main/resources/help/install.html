Installation Instructions

How to Run STED:

  • Download JDK 1.6.0 or Later.
  • Download the latest STED version.
  • Create a New folder and unzip sted.zip to create STED application.
  • Edit bin/sted.sh or bin/sted.bat and set the environment variables correctly for JAVA_HOME etc.,
  • Read readme.txt and follow instructions.
  • Read license.txt for License and Usage Terms.
  • Execute 'chmod +x sted.sh' in Unix OR the equivalent in Windows.
  • Run sted.sh in Linux OR sted.bat in Windows.
