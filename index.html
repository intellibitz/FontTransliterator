STED SARA Transliterator & Editor - An Open Source Transliterator and Editor for Any Language.

What STED does:

  • Helps Transliterate files between Any Language using Custom Character Mapping between Fonts.
  • Helps writing Native Language in Phonetic form using Standard Keyboard.
  • Helps maintain Custom Native Language Character Mapping/Encoding Scheme independent of any vendor.
  • Helps store files in ASCII (Phonetic) or in Native Language Characters or in both.
